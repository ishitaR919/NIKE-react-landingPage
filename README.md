**Cloning the Repository**

```bash
git clone https://github.com/adrianhajdin/nike_landing_page.git
cd nike_landing_page
```

**Installation**

Install the project dependencies using npm:

```bash
npm install
```


**Running the Project**

```bash
npm start
```
